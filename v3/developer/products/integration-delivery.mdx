**Integration Delivery** is Refold's product for one-off, scoped integration projects: connecting a specific customer's SAP, Workday, Salesforce, or NetSuite instance to your platform under a defined statement of work. You run the project lifecycle on Refold (FSD, TSD, field mapping, workflow generation, testing, go-live), and the resulting integration runs on the same execution platform as your Native integrations.

Use Integration Delivery when an integration is bespoke per customer and needs project management, not a self-serve toggle. For self-serve integrations your customers turn on, use [Native](/v3/developer/products/native).

## What you can do

- **Manage the full project lifecycle** from scoping through go-live, with first-class FSD (functional spec) and TSD (technical spec) documents on the project page.
- **Map fields visually** between source and destination systems and generate the workflow scaffolding from the mapping.
- **Test against sandbox tenants** and promote to production with one click when the project is approved.
- **Operate post-go-live**: run the integration on Refold's execution platform with the same logs, alerts, and reliability tooling as Native.
