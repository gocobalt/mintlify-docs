**MCP** is Refold's product for connecting AI agents to enterprise systems. You configure a Model Context Protocol server in the Refold dashboard, attach the apps your customers use, and hand them a URL. Their agent connects to that URL and gets authenticated, audited access to integration actions across Salesforce, SAP, NetSuite, Workday, and 300+ other applications.

No custom integration code, no agent framework lock-in. Refold handles authentication, execution, tenant isolation, and audit logging per request.

## What you can do

- **Create MCP servers** that expose a curated set of apps, actions, and workflows to AI agents.
- **Pick a tool mode**: **Direct** exposes each integration action as its own tool; **Agent** uses two meta-tools (RESOLVE and EXECUTE) so the agent discovers intent before calling.
- **Isolate every tenant**: Refold validates identity per request, so a single MCP URL safely serves many customers.
- **Audit every call**: see who called what, when, with what arguments, and what came back in [MCP Logs](/v3/mcp-ai-agents/security/mcp-logs).
