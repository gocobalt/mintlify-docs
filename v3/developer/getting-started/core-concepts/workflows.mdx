## The Workflow Builder

You build workflows using the visual Workflow Builder in the Refold dashboard. The builder provides a drag-and-drop canvas where you connect nodes, configure their inputs, and test the entire flow.

Refold provides over 20 node types for different operations: making HTTP requests, running custom code, looping over arrays, branching with conditional logic, handling files, and more. You can also use AI-powered workflow generation to build workflows from natural language descriptions.

## Key Characteristics

| Characteristic | Description |
|---------------|-------------|
| **DAG structure** | Nodes execute in a defined order with no circular dependencies |
| **Per-user execution** | Each workflow runs in the context of a specific Linked Account |
| **Trigger-driven** | Workflows start from events, webhooks, schedules, or API calls |
| **Reusable across users** | Build once, deploy for all your Linked Accounts |
| **Testable** | Test individual nodes or the entire workflow before deploying |

<Info>
Workflows are central to everything in Refold. Whether you are syncing CRM records, processing webhook payloads, or orchestrating multi-step business processes, you will build a workflow to do it.
</Info>
