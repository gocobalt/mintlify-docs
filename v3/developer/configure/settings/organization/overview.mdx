The **Overview** page holds the high-level metadata for a workspace: its
display name, the URL slug used in dashboard links, the workspace ID used in
API calls, the contact email, the logo, and the subscription tier.

Open it from **Settings > Organization > Overview**.

## Edit workspace details

<Steps>
  <Step title="Open Overview">
    Go to **Settings > Organization > Overview**.
  </Step>
  <Step title="Update fields">
    Edit any of **Name**, **Website**, or **Logo** (paste a public image URL).
  </Step>
  <Step title="Save">
    Click **Save**. Changes apply across the workspace immediately.
  </Step>
</Steps>

## Fields explained

| Field | What it does |
|---|---|
| **Name** | Display name shown in the dashboard, in invitation emails, and on the workspace switcher. |
| **Website** | Public URL of your company. Used in customer-facing flows like the Connect Portal. |
| **Email** | Workspace contact address. Refold sends billing and admin notifications here. To change it, click **Contact support**. |
| **Logo** | Public URL of an image used as the workspace avatar wherever the workspace name appears. |
| **Slug** | URL-safe identifier used in dashboard URLs (`refold-8301`). Read-only; copy it with the icon at the right of the field. |
| **ID** | Internal workspace identifier used in API requests and webhook payloads. Read-only; copy with the icon at the right of the field. |

## Subscription tier

The badge under the workspace logo in the right column shows your tier
(for example, **Enterprise**). To change tiers, contact your account
representative or use **Contact support** under the email field.
