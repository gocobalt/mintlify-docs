---
title: Create Hosted URL
api: post /api/v2/public/connect-url
description: Generate a hosted portal URL for your linked accounts to authenticate with third-party applications.
---

Generate a hosted URL that redirects users to Refold Connect at `https://connect.gocobalt.io`. The hosted portal provides a no-code solution for handling integration authentication and configuration without building your own UI.

The hosted portal handles:
- Listing all enabled applications
- Managing OAuth and key-based authentication flows
- Integration configuration for end users

<Note>
This API uses Session Token authentication instead of API Key authentication.
</Note>

## Authentication

<ParamField header="Authorization" type="string" required>
  Session token as Bearer token. Generate a session token using the [Generate Session Token](/v3/api-reference/session-token/generate-session-token) endpoint.
  
  **Format:** `Bearer {session_token}`
</ParamField>

## Query Parameters

<ParamField query="config_id" type="string">
  Config ID to associate with the hosted URL. If the config_id does not exist, a new config is created automatically.
  
  **Example:** `config_12345`
</ParamField>

## Body Parameters

<ParamField body="color" type="string">
  Text color for the hosted portal in hex format.
  
  **Example:** `#000000`, `#333333`
</ParamField>

<ParamField body="bgColor" type="string">
  Background color for the hosted portal in hex format.
  
  **Example:** `#FFFFFF`, `#F5F5F5`
</ParamField>

<ParamField body="name" type="string">
  Customer name to display in the hosted portal.
  
  **Example:** `Acme Corporation`
</ParamField>

## Example: Generate Hosted URL
```javascript
async function createHostedUrl(sessionToken, options = {}) {
  const params = new URLSearchParams();
  if (options.configId) params.append('config_id', options.configId);

  const url = `https://api.gocobalt.io/api/v2/public/connect-url${params.toString() ? '?' + params : ''}`;

  const response = await fetch(url, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${sessionToken}`
    },
    body: JSON.stringify({
      color: options.color || '#333333',
      bgColor: options.bgColor || '#FFFFFF',
      name: options.name || 'Customer'
    })
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const result = await createHostedUrl('your_session_token', {
  color: '#1a73e8',
  bgColor: '#f8f9fa',
  name: 'Acme Corporation'
});

console.log(`Hosted URL: ${result.hosted_url}`);
// Redirect user to this URL or open in new tab
window.open(result.hosted_url, '_blank');
```

## Example: Full Authentication Flow
```javascript
async function getHostedAuthUrl(apiKey, linkedAccountId, customization = {}) {
  // Step 1: Generate session token
  const tokenResponse = await fetch(
    'https://api.gocobalt.io/api/v2/public/session-token',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': apiKey
      },
      body: JSON.stringify({
        linked_account_id: linkedAccountId
      })
    }
  );

  if (!tokenResponse.ok) {
    throw new Error('Failed to generate session token');
  }

  const { token: sessionToken } = await tokenResponse.json();

  // Step 2: Generate hosted URL
  const hostedResponse = await fetch(
    'https://api.gocobalt.io/api/v2/public/connect-url',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${sessionToken}`
      },
      body: JSON.stringify({
        color: customization.textColor || '#000000',
        bgColor: customization.backgroundColor || '#FFFFFF',
        name: customization.customerName || 'Customer'
      })
    }
  );

  if (!hostedResponse.ok) {
    const error = await hostedResponse.json();
    throw new Error(error.message);
  }

  return hostedResponse.json();
}

// Usage
const { hosted_url } = await getHostedAuthUrl(
  'your_api_key',
  'user_12345',
  {
    textColor: '#1a73e8',
    backgroundColor: '#f8f9fa',
    customerName: 'John Smith'
  }
);

// Redirect user to authenticate
window.location.href = hosted_url;
```

## Example: Embed in iframe
```javascript
async function embedHostedPortal(containerId, sessionToken, options = {}) {
  const result = await createHostedUrl(sessionToken, options);

  const container = document.getElementById(containerId);
  const iframe = document.createElement('iframe');
  
  iframe.src = result.hosted_url;
  iframe.style.width = '100%';
  iframe.style.height = '600px';
  iframe.style.border = 'none';
  iframe.style.borderRadius = '8px';
  
  container.appendChild(iframe);
  
  return iframe;
}

// Usage
const iframe = await embedHostedPortal('integration-container', 'your_session_token', {
  name: 'Acme Corp',
  color: '#333333',
  bgColor: '#FFFFFF'
});
```

## Customization Options

| Property | Description | Example |
|----------|-------------|---------|
| `color` | Text/theme color of the portal | `#1a73e8` (blue) |
| `bgColor` | Background color of the portal | `#f8f9fa` (light gray) |
| `name` | Customer name displayed in the portal | `Acme Corporation` |

<Warning>
Session tokens expire after 24 hours. Generate a new token for each user session or before the token expires.
</Warning>
