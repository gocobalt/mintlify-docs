---
title: Get Execution by ID
api: get /api/v2/public/execution/:execution_id
description: Retrieve detailed information about a specific workflow execution.
---

Retrieve metadata and details about a specific workflow execution, including the associated workflow, application, status, and timestamps.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
</ParamField>

## Path Parameters

<ParamField path="execution_id" type="string" required>
  The unique identifier for the execution.
  
  **Example:** `65410cf6cdf931a12d7ab009`
</ParamField>

## Example: Get Execution Details
```javascript
async function getExecution(apiKey, linkedAccountId, executionId) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/execution/${executionId}`,
    {
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const execution = await getExecution(
  'your_api_key',
  'user_12345',
  '65410cf6cdf931a12d7ab009'
);

console.log(`Workflow: ${execution.associated_workflow.name}`);
console.log(`Application: ${execution.associated_application.name}`);
console.log(`Status: ${execution.status}`);
console.log(`Environment: ${execution.environment}`);
```

## Example: Track Execution Status
```javascript
async function trackExecution(apiKey, linkedAccountId, executionId) {
  const execution = await getExecution(apiKey, linkedAccountId, executionId);
  
  const statusEmoji = {
    COMPLETED: '✅',
    RUNNING: '⏳',
    ERRORED: '❌'
  };

  return {
    id: execution._id,
    status: execution.status,
    statusDisplay: `${statusEmoji[execution.status] || '❓'} ${execution.status}`,
    workflow: execution.associated_workflow.name,
    application: execution.associated_application.name,
    startedAt: new Date(execution.createdAt),
    environment: execution.environment
  };
}

// Usage
const status = await trackExecution('your_api_key', 'user_12345', '65410cf6cdf931a12d7ab009');
console.log(`${status.statusDisplay} - ${status.workflow} (${status.application})`);
```
