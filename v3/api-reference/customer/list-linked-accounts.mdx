---
title: List Linked Accounts
api: get /api/v2/public/linked-account
description: Retrieve a paginated list of all linked accounts in your organization.
---

Retrieve a paginated list of all linked accounts in your organization. Use query parameters to control pagination.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number to retrieve.
  
  **Example:** `1`, `2`, `3`
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of records per page. Maximum 100.
  
  **Example:** `10`, `25`, `50`
</ParamField>

## Example: Paginate Through All Accounts
```javascript
async function listLinkedAccounts(page, limit, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/linked-account?page=${page}&limit=${limit}`,
    {
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Get all accounts with pagination
async function getAllLinkedAccounts(apiKey) {
  const allAccounts = [];
  let page = 1;
  let hasMore = true;

  while (hasMore) {
    const result = await listLinkedAccounts(page, 50, apiKey);
    allAccounts.push(...result.docs);
    hasMore = result.hasNextPage;
    page++;
  }

  return allAccounts;
}

const accounts = await getAllLinkedAccounts('your_api_key');
console.log(`Total accounts: ${accounts.length}`);
```
