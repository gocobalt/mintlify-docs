---
title: Create Linked Account
api: post /api/v2/public/linked-account
description: Create a new linked account for your customer.
---

Create a new linked account to represent one of your customers. The linked account ID should be a unique identifier from your system, such as a user ID or account ID.

<Info>
If an account with the same `linked_account_id` already exists, this endpoint returns an error. Use the [Upsert Linked Account](/v3/api-reference/customer/upsert-a-linked-account) endpoint to create or update.
</Info>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings →  Credentials**.
</ParamField>

## Body Parameters

<ParamField body="linked_account_id" type="string" required>
  Unique identifier for the linked account. Typically matches an ID from your internal system.
  
  **Example:** `user_12345`, `account:98765`, `markwood@example.com`
</ParamField>

<ParamField body="name" type="string">
  Display name for the linked account.
  
  **Example:** `Acme Corporation`
</ParamField>

<ParamField body="your_app" type="object">
  Credentials for your application to receive data from Refold workflows.
  
  <Warning>
  If you include `your_app`, you must also include `auth_credentials`. An empty `your_app` object causes an error.
  </Warning>
  
  <Expandable title="properties">
    <ResponseField name="auth_credentials" type="object">
      Key-value pairs of credentials for authenticating with your APIs.
      <Expandable title="example">
        <ResponseField name="x-api-key" type="string">
          Your API key
        </ResponseField>
        <ResponseField name="base-url" type="string">
          Your API base URL
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField body="udf" type="object">
  User-defined fields to store custom data with the linked account.
  
  **Example:** `{ "plan": "enterprise", "region": "us-west" }`
</ParamField>

## Example: Create Linked Account
```javascript
async function createLinkedAccount(accountId, name, udf, apiKey) {
  const response = await fetch(
    'https://api.gocobalt.io/api/v2/public/linked-account',
    {
      method: 'POST',
      headers: {
        'x-api-key': apiKey,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        linked_account_id: accountId,
        name: name,
        udf: udf
      })
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const account = await createLinkedAccount(
  'user_12345',
  'Acme Corporation',
  { plan: 'enterprise', region: 'us-west' },
  'your_api_key'
);
console.log(`Created account: ${account.account_id}`);
```
