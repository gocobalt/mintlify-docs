---
title: Delete Linked Account
api: delete /api/v2/public/linked-account/:linked_account_id
description: Permanently delete a linked account and all associated data.
---

Permanently delete a linked account and all its associated data, including configurations, connections, and execution history.

<Warning>
This action is permanent and cannot be undone. All data associated with this linked account will be deleted, including integration connections, workflow configurations, and execution history.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="linked_account_id" type="string" required>
  The unique identifier of the linked account to delete.
  
  **Example:** `user_12345`
</ParamField>

## Example: Delete Linked Account with Confirmation
```javascript
async function deleteLinkedAccount(linkedAccountId, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/linked-account/${linkedAccountId}`,
    {
      method: 'DELETE',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Delete with confirmation
async function removeLinkedAccount(linkedAccountId, apiKey) {
  // First, verify the account exists
  const accountResponse = await fetch(
    `https://api.gocobalt.io/api/v2/public/linked-account/${linkedAccountId}`,
    { headers: { 'x-api-key': apiKey } }
  );

  if (!accountResponse.ok) {
    throw new Error('Account not found');
  }

  const account = await accountResponse.json();
  console.log(`Deleting account: ${account.name} (${account.account_id})`);

  // Delete the account
  const result = await deleteLinkedAccount(linkedAccountId, apiKey);
  console.log(result.message);
  
  return result;
}

await removeLinkedAccount('user_12345', 'your_api_key');
```
