---
title: Get Linked Account
api: get /api/v2/public/linked-account/:linked_account_id
description: Retrieve details of a specific linked account by its ID.
---

Retrieve information about a specific linked account using its unique identifier. Returns the account details including name, environment, user-defined fields, and timestamps.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345` or `markwood@example.com`
</ParamField>

## Example: Fetch Linked Account Details
```javascript
async function getLinkedAccount(linkedAccountId, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/linked-account/${linkedAccountId}`,
    {
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const account = await getLinkedAccount('user_12345', 'your_api_key');
console.log(`Account: ${account.name}, Environment: ${account.environment}`);
```
