---
title: Delete Config
api: delete /api/v2/public/slug/:slug/config/:config_id
description: Remove a configuration for a linked account and application.
---

Delete a configuration for a linked account. This removes all workflow settings and field values associated with the config. Use this when a user disconnects an integration or you need to reset their configuration.

<Warning>
This action is permanent and cannot be undone. All workflow settings and field values for this config will be lost.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

<ParamField path="config_id" type="string">
  The config identifier. If not provided, defaults to the `linked_account_id`.
  
  **Example:** `config_hubspot_001`
</ParamField>

## Example: Delete Config with Confirmation
```javascript
async function deleteConfig(linkedAccountId, slug, configId, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/slug/${slug}/config/${configId || ''}`,
    {
      method: 'DELETE',
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return { success: true, slug, configId };
}

// Usage with confirmation
async function disconnectIntegration(linkedAccountId, slug, configId, apiKey) {
  // First, disable all workflows
  await fetch('https://api.gocobalt.io/api/v2/public/config', {
    method: 'PUT',
    headers: {
      'x-api-key': apiKey,
      'linked_account_id': linkedAccountId,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      slug,
      config_id: configId,
      workflows: [] // Disable all workflows
    })
  });

  // Then delete the config
  return deleteConfig(linkedAccountId, slug, configId, apiKey);
}
```
