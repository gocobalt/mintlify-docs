---
title: Get Config
api: get /api/v2/public/slug/:slug/config/:config_id
description: Retrieve a specific config by its ID for an application.
---

Retrieve a specific configuration by its ID. Use this to check the current state of a config, including which workflows are enabled and what field values are set.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

<ParamField path="config_id" type="string">
  The config identifier. If not provided, defaults to the `linked_account_id`.
  
  **Example:** `config_hubspot_001`
</ParamField>

## Example: Check Workflow Status
```javascript
async function getWorkflowStatus(linkedAccountId, slug, configId, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/slug/${slug}/config/${configId || ''}`,
    {
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    throw new Error('Config not found');
  }

  const config = await response.json();
  
  return {
    configId: config.config_id,
    workflows: config.workflows.map(w => ({
      id: w.id,
      name: w.name,
      enabled: w.enabled,
      configuredFields: w.fields.filter(f => f.value).length,
      requiredFields: w.fields.filter(f => f.required).length
    })),
    isFullyConfigured: config.workflows.every(w => 
      !w.fields.some(f => f.required && !f.value)
    )
  };
}
```
