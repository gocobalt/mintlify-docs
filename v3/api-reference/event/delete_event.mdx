---
title: 'Delete event'
api: delete /api/v2/public/event/trigger/:event_id
---

This API deletes a Cobalt [event](https://app.gocobalt.io/your-app#:~:text=Add%20events%20to%20send%20data%20from%20your%20app%20to%20other%20apps) from your org's app.

## Request description
#### Path Parameters
<ParamField path="event_id" type="string" required>
  Event Id. For Eg: 653b8710aa28576f28afd0cb
</ParamField>

## Response description

<Tabs>
  <Tab title="200">
    <ResponseField name="_id" type="string">
      Event Id
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the event
    </ResponseField>
    <ResponseField name="response" type="array">
      Array of response field objects
      <Expandable title="properties">
        <ResponseField name="key" type="string">
          Field key name
        </ResponseField>
        <ResponseField name="type" type="string">
          Field data type
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="raw_response" type="string">
      Raw response string of the event model
    </ResponseField>
  </Tab>
  <Tab title="500">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="error" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
    <ResponseField name="request_id" type="string">
    System generated request id
    </ResponseField>
  </Tab>
</Tabs>
