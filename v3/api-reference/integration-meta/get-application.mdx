---
title: Get Application
api: get /api/v2/public/application/:slug
description: Retrieve details of a specific application by its slug.
---

Retrieve detailed information about a single application by its slug. Returns the same shape as [List Applications](/v3/api-reference/integration-meta/list-applications), but as a single object — including connection status, authentication requirements, and connected-account details for the linked account.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.

  **Example:** `cobalt_test_user`
</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>
  The application slug.

  **Example:** `hubspot`, `slack`, `salesforce`
</ParamField>

## Example: Check connection status
```javascript
async function getApplication(slug, linkedAccountId, apiKey) {
  const response = await fetch(
    `https://app.refold.ai/api/v2/public/application/${slug}`,
    {
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: check if HubSpot is connected
const hubspot = await getApplication('hubspot', 'cobalt_test_user', 'YOUR_API_KEY');

if (hubspot.connected) {
  console.log(`Connected at: ${hubspot.connected_accounts[0].connectedAt}`);
} else {
  console.log('HubSpot is not connected');
}

if (hubspot.reauth_required) {
  console.log('Warning: re-authentication required');
}
```

## Example: Build an auth form for key-based apps
```javascript
async function getAuthFields(slug, linkedAccountId, apiKey) {
  const app = await getApplication(slug, linkedAccountId, apiKey);

  if (app.auth_type !== 'keybased') {
    return { type: 'oauth', app };
  }

  return {
    type: 'keybased',
    app,
    fields: (app.auth_input_map || []).map(field => ({
      name: field.name,
      label: field.label,
      placeholder: field.placeholder,
      required: field.required,
      inputType: field.type
    }))
  };
}

const authInfo = await getAuthFields('custom-testapp', 'cobalt_test_user', 'YOUR_API_KEY');
if (authInfo.type === 'keybased') {
  authInfo.fields.forEach(f => console.log(`- ${f.label} (${f.required ? 'required' : 'optional'})`));
}
```
