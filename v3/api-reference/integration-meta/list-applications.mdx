---
title: List Applications
api: get /api/v2/public/application
description: Retrieve all enabled applications for a linked account.
---

Retrieve a list of all applications enabled for your organization and their connection status for a specific linked account. Use this to build custom integration UIs showing which apps are available and connected.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

## Example: Build Integration Catalog
```javascript
async function listApplications(linkedAccountId, apiKey) {
  const response = await fetch(
    'https://api.gocobalt.io/api/v2/public/application',
    {
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Get all apps and categorize
const apps = await listApplications('user_12345', 'your_api_key');

const connectedApps = apps.filter(app => app.connected);
const availableApps = apps.filter(app => !app.connected);

console.log(`Connected: ${connectedApps.length}, Available: ${availableApps.length}`);

// Group by category
const byCategory = apps.reduce((acc, app) => {
  const category = app.tags[0] || 'Other';
  acc[category] = acc[category] || [];
  acc[category].push(app);
  return acc;
}, {});

console.log('Apps by category:', Object.keys(byCategory));
```

## Example: Check Re-authentication Requirements
```javascript
async function getAppsNeedingReauth(linkedAccountId, apiKey) {
  const apps = await listApplications(linkedAccountId, apiKey);
  
  return apps.filter(app => app.connected && app.reauth_required);
}

// Usage: Alert user about expired connections
const reauthApps = await getAppsNeedingReauth('user_12345', 'your_api_key');

if (reauthApps.length > 0) {
  console.log('The following apps need re-authentication:');
  reauthApps.forEach(app => console.log(`- ${app.name} (${app.slug})`));
}
```
