---
title: Delete Config
api: delete /api/v2/public/slug/{slug}/config
description: Delete the installed config for an app (config ID defaulted).
---

Delete the config installation for an app on a linked account, without specifying a `config_id` — it defaults to the linked account ID. To target a specific config ID, use [Delete Config by ID](/v3/api-reference/configs/delete-config).

## Authentication

Configs are scoped to a linked account via the `linked_account_id` header.

<ParamField header="x-api-key" type="string" required>Your Refold API key.</ParamField>
<ParamField header="linked_account_id" type="string" required>The linked account.</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>The app slug. **Example:** `hubspot`</ParamField>
