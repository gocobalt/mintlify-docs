---
title: Delete Config by ID
openapi: "DELETE /api/v2/public/slug/{slug}/config/{configId}"
description: Delete a config installation by config ID.
---

Delete a config installation for an app and a specific `config_id`. (To delete the default config for a linked account, omit the ID — see [Delete Config](/v3/api-reference/configs/delete-config-default).)

<ResponseExample>
```json 200
{
  "message": "Installation Deleted Success"
}
```
```json 404
{
  "status_code": 404,
  "http_error_type": "NOT_FOUND",
  "error_code": "SERVER_ERROR",
  "message": "Config with config id user_12345 not found"
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl -X DELETE "https://app.refold.ai/api/v2/public/slug/{slug}/config/{config_id}" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID"
```
</RequestExample>
