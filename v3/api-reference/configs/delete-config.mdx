---
title: Delete Config by ID
api: delete /api/v2/public/slug/{slug}/config/{config_id}
description: Delete a config installation by config ID.
---

Delete a config installation for an app and a specific `config_id`. (To delete the default config for a linked account, omit the ID — see [Delete Config](/v3/api-reference/configs/delete-config-default).)

## Authentication

Configs are scoped to a linked account via the `linked_account_id` header.

<ParamField header="x-api-key" type="string" required>Your Refold API key.</ParamField>
<ParamField header="linked_account_id" type="string" required>The linked account.</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>The app slug. **Example:** `hubspot`</ParamField>
<ParamField path="config_id" type="string" required>The config ID to delete.</ParamField>
