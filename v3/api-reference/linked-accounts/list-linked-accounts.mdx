---
title: List Linked Accounts
openapi: "GET /api/v2/public/linked-account"
description: Retrieve a paginated list of all linked accounts in your organization.
---

Retrieve a paginated list of the linked accounts in your organization and environment (derived from your API key). Each account's sensitive fields (`integrations[*].credentials`, `authTokens`, and `connections`) are stripped from the response.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number to retrieve.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of linked accounts per page.
</ParamField>

<ParamField query="env_variable_name" type="string">
  Filter accounts by an environment-variable name set on the account.
</ParamField>

<ParamField query="env_variable_value" type="string">
  Filter accounts by an environment-variable value. Use together with `env_variable_name`.
</ParamField>

## Response

        <ResponseField name="udf" type="array">User-defined fields as `{key_name, key_value}` pairs.</ResponseField>
