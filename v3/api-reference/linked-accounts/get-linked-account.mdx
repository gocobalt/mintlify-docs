---
title: Get Linked Account
api: get /api/v2/public/linked-account/:linked_account_id
description: Retrieve details of a specific linked account by its ID.
---

Retrieve a single linked account by its `account_id`. Sensitive fields (`integrations[*].credentials`, `authTokens`, and `connections`) are stripped from the response.

## Authentication

Reading linked-account records is an org-level operation, so this endpoint uses API-key auth.

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="linked_account_id" type="string" required>
  Your unique identifier for the account.

  **Example:** `user_12345`
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="_id" type="string">Internal record ID.</ResponseField>
    <ResponseField name="associated_org" type="string">The org that owns this account.</ResponseField>
    <ResponseField name="account_id" type="string">Your unique identifier for the account.</ResponseField>
    <ResponseField name="environment" type="string">Environment: `test` or `live`.</ResponseField>
    <ResponseField name="name" type="string">Display name.</ResponseField>
    <ResponseField name="udf" type="array">
      User-defined fields.
      <Expandable title="properties">
        <ResponseField name="key_name" type="string">Field key.</ResponseField>
        <ResponseField name="key_value" type="string">Field value.</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="integrations" type="array">
      Connected integrations (credentials/tokens stripped).
      <Expandable title="properties">
        <ResponseField name="type" type="string">App type, e.g. `hubspot`, `custom`.</ResponseField>
        <ResponseField name="name" type="string">App display name.</ResponseField>
        <ResponseField name="app_id" type="string | null">Custom-app ID, or `null` for native apps.</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 creation timestamp.</ResponseField>
    <ResponseField name="updatedAt" type="string">ISO 8601 last-update timestamp.</ResponseField>
  </Tab>

  <Tab title="404 Not Found">

  </Tab>
</Tabs>
