---
title: Delete Field Value
api: delete /api/v2/public/config/field/:fieldId
description: Clear the value of a specific config field.
---

Delete (clear) the value of a specific configuration field. This resets the field to its default state, removing any user-set value. The field definition itself remains; only the value is cleared.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

<ParamField header="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

## Path Parameters

<ParamField path="fieldId" type="string" required>
  The unique identifier of the field to clear.
  
  **Example:** `670fcb9e023ebc1f370c5139`
</ParamField>

## Query Parameters

<ParamField query="workflow_id" type="string">
  Filter to a specific workflow's field. Required when deleting a workflow-specific field value.
  
  **Example:** `67fbc1ec954bde0dcae218c8`
</ParamField>

## Example: Reset Field Value
```javascript
async function resetFieldValue(linkedAccountId, slug, fieldId, workflowId, apiKey) {
  const url = new URL(`https://api.gocobalt.io/api/v2/public/config/field/${fieldId}`);
  if (workflowId) {
    url.searchParams.set('workflow_id', workflowId);
  }

  const response = await fetch(url, {
    method: 'DELETE',
    headers: {
      'x-api-key': apiKey,
      'linked_account_id': linkedAccountId,
      'slug': slug
    }
  });

  const field = await response.json();
  
  return {
    fieldId: field.id,
    fieldName: field.name,
    cleared: true
  };
}

// Usage: Reset a channel selection
await resetFieldValue(
  'user_12345',
  'slack',
  '64da0b57c9ae95561bb0a24d',
  null,
  'your_api_key'
);
```
