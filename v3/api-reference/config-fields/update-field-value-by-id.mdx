---
title: Update Field Value
api: put /api/v2/public/config/field/:fieldId
description: Update the value of a specific config field.
---

Update the value of a specific configuration field. Use this to set field values individually, which is useful for partial config updates or when building custom configuration interfaces.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

<ParamField header="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

## Path Parameters

<ParamField path="fieldId" type="string" required>
  The unique identifier of the field to update.
  
  **Example:** `670fcb9e023ebc1f370c5139`
</ParamField>

## Query Parameters

<ParamField query="workflow_id" type="string">
  Filter to a specific workflow's field. Required when updating a workflow-specific field.
  
  **Example:** `67fbc1ec954bde0dcae218c8`
</ParamField>

<ParamField query="config_id" type="string">
  The config identifier. Defaults to `linked_account_id` if not provided.
  
  **Example:** `config_hubspot_001`
</ParamField>

## Body Parameters

<ParamField body="slug" type="string" required>
  Application slug.
  
  **Example:** `hubspot`
</ParamField>

<ParamField body="config_id" type="string" required>
  The config identifier.
  
  **Example:** `user_12345`
</ParamField>

<ParamField body="fields" type="array" required>
  Array of field updates.
  <Expandable title="Field Object">
    <ParamField body="id" type="string" required>
      Unique field ID
    </ParamField>
    <ParamField body="name" type="string">
      Field display name
    </ParamField>
    <ParamField body="options" type="array">
      Updated options for `select` or `map` fields
      <Expandable title="Option Object">
        <ParamField body="name" type="string">
          Display name
        </ParamField>
        <ParamField body="value" type="string">
          Option value
        </ParamField>
      </Expandable>
    </ParamField>
    <ParamField body="labels" type="array">
      Updated labels for `map` fields
      <Expandable title="Label Object">
        <ParamField body="name" type="string">
          Label display name
        </ParamField>
        <ParamField body="value" type="string">
          Label value
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="workflows" type="array" required>
  Array of workflow field updates.
  <Expandable title="Workflow Object">
    <ParamField body="id" type="string">
      Workflow ID
    </ParamField>
    <ParamField body="name" type="string">
      Workflow name
    </ParamField>
    <ParamField body="fields" type="array">
      Workflow-specific field updates (same structure as global fields)
    </ParamField>
  </Expandable>
</ParamField>

## Example: Update Field Mapping
```javascript
async function updateFieldMapping(linkedAccountId, slug, fieldId, mappings, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/config/field/${fieldId}`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId,
        'slug': slug,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        slug,
        config_id: linkedAccountId,
        fields: [
          {
            id: fieldId,
            options: mappings.map(m => ({
              name: m.displayName,
              value: m.fieldValue
            }))
          }
        ],
        workflows: []
      })
    }
  );

  return response.json();
}

// Usage: Update contact field mappings
await updateFieldMapping(
  'user_12345',
  'hubspot',
  '670fcb9e023ebc1f370c5139',
  [
    { displayName: 'Customer Name', fieldValue: 'firstname' },
    { displayName: 'Customer Email', fieldValue: 'email' },
    { displayName: 'Customer Company', fieldValue: 'company' }
  ],
  'your_api_key'
);
```
