---
title: Get Rule Engine Options
openapi: "POST /api/v2/public/config/rule-engine/{fieldId}"
description: Resolve column options for a rule-engine config field.
---

Resolve the available column options for a rule-engine config field. Provide the field-shaped `rule_column` describing the column whose options you want. The field must be of rule-engine type with defined columns.

<ResponseExample>
```json 200
{
  "options": [
    { "name": "Stage", "value": "stage" },
    { "name": "Amount", "value": "amount" }
  ]
}
```
```json 400
{
  "status_code": 400,
  "http_error_type": "BAD_REQUEST",
  "error_code": "SERVER_ERROR",
  "message": "Field with id 699fec7a5bd27ed2af7243ad is not of type rule engine or has undefined columns"
}
```
</ResponseExample>

<RequestExample>
```bash cURL (API key)
curl -X POST "https://app.refold.ai/api/v2/public/config/rule-engine/{fieldId}" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID" \
  -H "slug: {slug}" \
  -H "Content-Type: application/json" \
  -d '{ "rule_column": { "name": "Stage", "field_type": "select" } }'
```
```bash cURL (session token)
curl -X POST "https://app.refold.ai/api/v2/public/config/rule-engine/{fieldId}" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \
  -H "slug: {slug}" \
  -H "Content-Type: application/json" \
  -d '{ "rule_column": { "name": "Stage", "field_type": "select" } }'
```
</RequestExample>
