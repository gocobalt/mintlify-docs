---
title: Get Field by ID
api: post /api/v2/public/config/field/:fieldId
description: Retrieve a specific config field with its options and current value.
---

Retrieve details for a specific configuration field by its ID. Use this to get the current options, labels, and value for a single field, which is useful for dynamic forms or when you need to refresh field options.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

<ParamField header="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

## Path Parameters

<ParamField path="fieldId" type="string" required>
  The unique identifier of the field.
  
  **Example:** `670fcb9e023ebc1f370c5139`
</ParamField>

## Query Parameters

<ParamField query="workflow_id" type="string">
  Filter to a specific workflow's field. Required when the field belongs to a workflow.
  
  **Example:** `67fbc1ec954bde0dcae218c8`
</ParamField>

<ParamField query="config_id" type="string">
  The config identifier. Defaults to `linked_account_id` if not provided.
  
  **Example:** `config_hubspot_001`
</ParamField>

## Example: Refresh Field Options
```javascript
async function getFieldOptions(linkedAccountId, slug, fieldId, workflowId, apiKey) {
  const url = new URL(`https://api.gocobalt.io/api/v2/public/config/field/${fieldId}`);
  if (workflowId) {
    url.searchParams.set('workflow_id', workflowId);
  }

  const response = await fetch(url, {
    method: 'POST',
    headers: {
      'x-api-key': apiKey,
      'linked_account_id': linkedAccountId,
      'slug': slug
    }
  });

  const field = await response.json();
  
  return {
    id: field.id,
    name: field.name,
    type: field.field_type,
    required: field.required,
    options: field.options,
    labels: field.labels,
    hasDynamicFields: field.has_dynamic_map_fields,
    isHidden: field.hidden
  };
}

// Usage: Refresh channel options for Slack
const channelField = await getFieldOptions(
  'user_12345',
  'slack',
  '64da0b57c9ae95561bb0a24d',
  null,
  'your_api_key'
);

console.log('Available channels:', channelField.options);
```
