---
title: Clear Config Field Value
openapi: "DELETE /api/v2/public/config/field/{fieldId}"
description: Clear the value of a single config field.
---

Clear the value of a single config field, optionally scoped to a specific workflow. The field returns to its unset/default state.

<ResponseExample>
```json 200
{
  "id": "699fec7a5bd27ed2af7243ad",
  "name": "Select",
  "field_type": "select",
  "multiple": true,
  "options": [
    { "name": "A", "value": "A" },
    { "name": "B", "value": "B" },
    { "name": "C", "value": "C" }
  ],
  "defaultValue": ""
}
```
```json 404
{
  "status_code": 404,
  "http_error_type": "NOT_FOUND",
  "error_code": "SERVER_ERROR",
  "message": "Config with config id user_12345 not found"
}
```
</ResponseExample>

<RequestExample>
```bash cURL (API key)
curl -X DELETE "https://app.refold.ai/api/v2/public/config/field/{fieldId}" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID" \
  -H "slug: {slug}"
```
```bash cURL (session token)
curl -X DELETE "https://app.refold.ai/api/v2/public/config/field/{fieldId}" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \
  -H "slug: {slug}"
```
</RequestExample>
