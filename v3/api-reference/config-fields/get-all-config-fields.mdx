---
title: List Config Fields
api: post /api/v2/public/config/field
description: Retrieve all fields in a config for an application.
---

Retrieve all configuration fields for an application, including both global fields and workflow-specific fields. Use this to build custom configuration UIs or to check the current state of all fields.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

<ParamField header="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `slack`
</ParamField>

## Example: Build Dynamic Field Form
```javascript
async function getAllConfigFields(linkedAccountId, slug, apiKey) {
  const response = await fetch('https://api.gocobalt.io/api/v2/public/config/field', {
    method: 'POST',
    headers: {
      'x-api-key': apiKey,
      'linked_account_id': linkedAccountId,
      'slug': slug
    }
  });

  const config = await response.json();
  
  // Organize fields by type for UI rendering
  return {
    globalFields: config.fields.map(formatField),
    workflows: config.workflows.map(workflow => ({
      id: workflow.id,
      name: workflow.name,
      enabled: workflow.enabled,
      fields: workflow.fields.map(formatField)
    }))
  };
}

function formatField(field) {
  return {
    id: field.id,
    name: field.name,
    type: field.field_type,
    required: field.required,
    options: field.options || [],
    labels: field.labels || [],
    // Determine UI component type
    component: getComponentType(field.field_type)
  };
}

function getComponentType(fieldType) {
  const components = {
    'text': 'TextInput',
    'number': 'NumberInput',
    'select': 'Dropdown',
    'map': 'FieldMapper'
  };
  return components[fieldType] || 'TextInput';
}
```

## Field Types

| Type | Description | UI Component |
|------|-------------|--------------|
| `text` | Free-form text input | Text input |
| `number` | Numeric value | Number input |
| `select` | Single selection from options | Dropdown |
| `map` | Field mapping between systems | Mapping table |
