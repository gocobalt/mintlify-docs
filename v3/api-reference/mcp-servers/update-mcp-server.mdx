---
title: Update MCP Server
api: put /api/v2/public/mcp-servers/{mcp_server_id}
description: Update an MCP server's configuration.
---

Update an MCP server. This is patch-style — only the fields you send are written. Use it to attach apps, define chains (agents), add prompt suggestions, and set branding.

## Authentication

This endpoint is scoped to a linked account, so it accepts **either** authentication method.

**Option 1 — API key**

<ParamField header="x-api-key" type="string">Your Refold API key.</ParamField>
<ParamField header="linked_account_id" type="string">The linked account. Required with API-key auth.</ParamField>

**Option 2 — Session token**

<ParamField header="Authorization" type="string">Session token. **Format:** `Bearer YOUR_SESSION_TOKEN`</ParamField>

## Path Parameters

<ParamField path="mcp_server_id" type="string" required>The MCP server ID.</ParamField>

## Body Parameters

<ParamField body="name" type="string">Server name.</ParamField>
<ParamField body="description" type="string">Server description.</ParamField>
<ParamField body="associated_apps" type="array">Apps to expose, each an app object with its actions/workflows.</ParamField>
<ParamField body="chains" type="array">Chain (agent) definitions.</ParamField>
<ParamField body="suggestions" type="array">Prompt suggestions shown to users.</ParamField>
<ParamField body="agent_name" type="string">Display name for the agent.</ParamField>
<ParamField body="primary_color" type="string">Primary brand color (hex).</ParamField>
<ParamField body="secondary_color" type="string">Secondary brand color (hex).</ParamField>
