---
title: Create MCP Server
openapi: "POST /api/v2/public/mcp-servers"
description: Create a new MCP server for a linked account.
---

Create a new MCP (Model Context Protocol) server. Only `name` and `description` are required at creation; configure associated apps, chains, and suggestions afterward via [Update MCP Server](/v3/api-reference/mcp-servers/update-mcp-server).

<ResponseExample>
```json 201
{
  "_id": "6a314ee8d0aeb8165beacc6e",
  "name": "Production CRM Server",
  "description": "MCP server for CRM workflows"
}
```
```json 400
{
  "status_code": 400,
  "http_error_type": "BAD_REQUEST",
  "error_code": "SERVER_ERROR",
  "message": "name and description are required"
}
```
</ResponseExample>

<RequestExample>
```bash cURL (API key)
curl -X POST "https://app.refold.ai/api/v2/public/mcp-servers" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Production CRM Server", "description": "MCP server for CRM workflows" }'
```
```bash cURL (session token)
curl -X POST "https://app.refold.ai/api/v2/public/mcp-servers" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Production CRM Server", "description": "MCP server for CRM workflows" }'
```
</RequestExample>
