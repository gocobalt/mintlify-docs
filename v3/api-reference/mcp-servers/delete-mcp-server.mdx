---
title: Delete MCP Server
openapi: "DELETE /api/v2/public/mcp-servers/{mcp_server_id}"
description: Delete an MCP server.
---

Delete an MCP server. This removes the server and its configuration.

<ResponseExample>
```json 200
```
```json 404
{
  "status_code": 404,
  "http_error_type": "NOT_FOUND",
  "error_code": "SERVER_ERROR",
  "message": "MCPEntry not found"
}
```
</ResponseExample>

<RequestExample>
```bash cURL (API key)
curl -X DELETE "https://app.refold.ai/api/v2/public/mcp-servers/{mcp_server_id}" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID"
```
```bash cURL (session token)
curl -X DELETE "https://app.refold.ai/api/v2/public/mcp-servers/{mcp_server_id}" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"
```
</RequestExample>
