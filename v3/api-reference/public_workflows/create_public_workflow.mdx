---
title: Create Public Workflow
api: post /api/v2/public/workflow
description: Create a new public workflow for a linked account.
---

Create a new public workflow that linked accounts can customize and execute. Public workflows enable end-users to build their own automation workflows within your embedded integration experience.

<Info>
Public workflows are created with a default Start node and operate in the production environment. Users can edit workflows in draft mode before publishing.
</Info>

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token using your session token for authentication.
  
  **Format:** `Bearer YOUR_SESSION_TOKEN`
  
  Generate a session token using the [Session Token API](/v3/api-reference/session-token/generate-session-token).
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Display name for the workflow.
  
  **Example:** `Sync Contacts to CRM`
</ParamField>

<ParamField body="slug" type="string" required>
  Application slug the workflow is associated with.
  
  **Example:** `hubspot`, `salesforce`, `slack`
</ParamField>

<ParamField body="description" type="string">
  Description of what the workflow does.
  
  **Example:** `Automatically sync new contacts from our platform to HubSpot`
</ParamField>

## Example: Create Public Workflow
```javascript
async function createPublicWorkflow(sessionToken, workflowData) {
  const response = await fetch('https://api.gocobalt.io/api/v2/public/workflow', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${sessionToken}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(workflowData)
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Create a HubSpot sync workflow
const workflow = await createPublicWorkflow('session_token_here', {
  name: 'Sync Contacts to HubSpot',
  slug: 'hubspot',
  description: 'Automatically sync new contacts from our platform to HubSpot CRM'
});

console.log(`Workflow created: ${workflow._id}`);
console.log(`Embed URL: https://app.gocobalt.io/workflows/${workflow._id}?token=SESSION_TOKEN`);
```

## Example: Embed Workflow Editor
```javascript
// After creating a workflow, embed the editor in your application
function embedWorkflowEditor(workflowId, sessionToken) {
  const iframe = document.createElement('iframe');
  iframe.src = `https://app.gocobalt.io/workflows/${workflowId}?token=${sessionToken}`;
  iframe.style.width = '100%';
  iframe.style.height = '100%';
  iframe.style.border = 'none';
  
  document.getElementById('workflow-container').appendChild(iframe);
}

// Usage
embedWorkflowEditor(workflow._id, 'session_token_here');
```
