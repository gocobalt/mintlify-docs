---
title: Delete Schedule
openapi: "DELETE /api/v1/workflow/public/scheduled/{schedule_id}"
description: Delete a scheduled trigger.
---

Permanently delete a schedule and deregister its job. To temporarily stop a schedule instead, use [Pause Schedule](/v3/api-reference/schedule/pause-schedule).

## Authentication

<ParamField header="x-api-key" type="string" required>Your Refold API key.</ParamField>
<ParamField header="linked_account_id" type="string" required>The linked account that owns the schedule.</ParamField>

## Path Parameters

<ParamField path="schedule_id" type="string" required>The schedule ID to delete.</ParamField>
