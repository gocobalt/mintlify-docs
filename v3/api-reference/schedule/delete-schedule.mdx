---
title: Delete Schedule
api: delete /api/v1/workflow/public/scheduled/:schedule_id
description: Permanently delete a scheduled job.
---

Permanently delete a scheduled job. This action cannot be undone. The workflow will no longer execute on the previously configured schedule.

<Warning>
This action is permanent. To temporarily stop a schedule, use [Pause Schedule](/v3/api-reference/schedule/pause-schedule) instead.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="schedule_id" type="string" required>
  The unique identifier of the schedule to delete.
  
  **Example:** `67becf62f29636d291eaadc0`
</ParamField>

## Example: Delete Schedule
```javascript
async function deleteSchedule(apiKey, scheduleId) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v1/workflow/public/scheduled/${scheduleId}`,
    {
      method: 'DELETE',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const result = await deleteSchedule('your_api_key', '67becf62f29636d291eaadc0');
console.log(result.message); // "Scheduled Job Deleted Successfully"
```

## Example: Delete All Schedules for User
```javascript
async function deleteAllUserSchedules(apiKey, linkedAccountId) {
  // First, list all schedules for the user
  const schedules = await listSchedules(apiKey, { linkedAccountId, limit: 100 });
  
  const results = await Promise.allSettled(
    schedules.docs.map(schedule => deleteSchedule(apiKey, schedule._id))
  );

  const deleted = results.filter(r => r.status === 'fulfilled').length;
  const failed = results.filter(r => r.status === 'rejected').length;

  return { deleted, failed, total: schedules.docs.length };
}

// Usage
const result = await deleteAllUserSchedules('your_api_key', 'user_12345');
console.log(`Deleted ${result.deleted} of ${result.total} schedules`);
```
