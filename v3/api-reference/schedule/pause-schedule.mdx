---
title: Pause Schedule
api: put /api/v1/workflow/public/scheduled/:schedule_id/pause
description: Pause an active schedule to temporarily stop workflow executions.
---

Pause an active schedule to temporarily stop scheduled workflow executions. The schedule configuration is preserved and can be resumed later.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="schedule_id" type="string" required>
  The unique identifier of the schedule to pause.
  
  **Example:** `67becf62f29636d291eaadc0`
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    Returns the updated schedule object with `inactive` set to `true`.

    <ResponseField name="_id" type="string">
      Schedule identifier
    </ResponseField>

    <ResponseField name="inactive" type="boolean">
      Always `true` after pausing
    </ResponseField>

    <ResponseField name="workflow" type="string">
      Workflow ID or alias
    </ResponseField>

    <ResponseField name="linked_account_id" type="string">
      Linked account ID
    </ResponseField>

    <ResponseField name="cron_patten" type="string">
      Cron pattern (preserved)
    </ResponseField>

    <ResponseField name="options" type="object">
      Schedule options (preserved)
    </ResponseField>

    <ResponseField name="updatedAt" type="string">
      ISO 8601 timestamp of the pause action
    </ResponseField>
  </Tab>

  <Tab title="400 Bad Request">
    <ResponseField name="status_code" type="integer">
      HTTP status code
    </ResponseField>

    <ResponseField name="message" type="string">
      Error description (e.g., "Scheduled Job Already Inactive")
    </ResponseField>
  </Tab>

  <Tab title="404 Not Found">
    <ResponseField name="status_code" type="integer">
      HTTP status code
    </ResponseField>

    <ResponseField name="error_code" type="string">
      `RESOURCE_NOT_FOUND`
    </ResponseField>

    <ResponseField name="message" type="string">
      "Scheduled Job Not Found"
    </ResponseField>
  </Tab>
</Tabs>

## Example: Pause Schedule
```javascript
async function pauseSchedule(apiKey, scheduleId) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v1/workflow/public/scheduled/${scheduleId}/pause`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const schedule = await pauseSchedule('your_api_key', '67becf62f29636d291eaadc0');
console.log(`Schedule paused: ${schedule.inactive}`);
```
