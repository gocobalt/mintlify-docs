---
title: Update Schedule
api: put /api/v1/workflow/public/scheduled/:schedule_id
description: Update an existing schedule's trigger configuration and payload.
---

Update an existing schedule's trigger type, cron expression, interval dates, or payload. The schedule can be changed from cron to interval or vice versa.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="schedule_id" type="string" required>
  The unique identifier of the schedule to update.
  
  **Example:** `67becf62f29636d291eaadc0`
</ParamField>

## Body Parameters

<ParamField body="trigger_type" type="string" required>
  The type of schedule trigger.
  
  - `cron` - Use a cron expression
  - `interval` - Use specific date/time values
</ParamField>

<ParamField body="payload" type="object" required>
  Event payload to send during workflow execution.
</ParamField>

### Cron Trigger Parameters

<Info>
Required when `trigger_type` is `cron`.
</Info>

<ParamField body="regex" type="string">
  Cron expression. **Example:** `*/30 * * * *`
</ParamField>

<ParamField body="currentDate" type="string">
  Current date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="startDate" type="string">
  Start date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="endDate" type="string">
  End date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="tz" type="string">
  Timezone. **Example:** `America/New_York`
</ParamField>

### Interval Trigger Parameters

<Info>
Required when `trigger_type` is `interval`.
</Info>

<ParamField body="interval_dates" type="array">
  Array of specific date/time values. See [Register Schedule](/v3/api-reference/schedule/register-schedule) for format.
</ParamField>

## Example: Update Schedule
```javascript
async function updateSchedule(apiKey, scheduleId, config) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v1/workflow/public/scheduled/${scheduleId}`,
    {
      method: 'PUT',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': apiKey
      },
      body: JSON.stringify(config)
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Change schedule from every 30 minutes to daily at 9 AM
const today = new Date().toISOString().split('T')[0];

const updated = await updateSchedule('your_api_key', '67becf62f29636d291eaadc0', {
  trigger_type: 'cron',
  payload: { source: 'daily_sync' },
  regex: '0 9 * * *',
  currentDate: today,
  startDate: '2025-01-21',
  endDate: '2025-12-31',
  tz: 'America/New_York'
});

console.log(`Schedule updated: ${updated.cron_patten}`);
```
