---
title: Resume Schedule
api: put /api/v1/workflow/public/scheduled/:schedule_id/resume
description: Resume a paused schedule to restart workflow executions.
---

Resume a previously paused schedule to restart scheduled workflow executions using the original configuration.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="schedule_id" type="string" required>
  The unique identifier of the schedule to resume.
  
  **Example:** `67becf62f29636d291eaadc0`
</ParamField>

## Response

## Example: Resume Schedule
```javascript
async function resumeSchedule(apiKey, scheduleId) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v1/workflow/public/scheduled/${scheduleId}/resume`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const schedule = await resumeSchedule('your_api_key', '67becf62f29636d291eaadc0');
console.log(`Schedule resumed: ${!schedule.inactive}`);
```

## Example: Toggle Schedule State
```javascript
async function toggleSchedule(apiKey, scheduleId, currentState) {
  const action = currentState ? 'resume' : 'pause';
  
  const response = await fetch(
    `https://api.gocobalt.io/api/v1/workflow/public/scheduled/${scheduleId}/${action}`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Toggle from paused to active
const schedule = await toggleSchedule('your_api_key', '67becf62f29636d291eaadc0', true);
console.log(`Schedule is now ${schedule.inactive ? 'paused' : 'active'}`);
```
