---
title: List MCP Servers
api: get /api/v2/public/mcp-servers
description: Retrieve a paginated list of all MCP servers in your organization.
---

Retrieve all MCP servers configured for your organization. MCP (Model Context Protocol) servers enable AI agents to execute integration actions across enterprise platforms like Salesforce, NetSuite, and SAP.

<Note>
This API supports two authentication methods: Session Token OR API Key with linked account ID.
</Note>

## Authentication

### Option 1: Session Token

<ParamField header="Authorization" type="string" required>
  Bearer token using your session token.
  
  **Format:** `Bearer YOUR_SESSION_TOKEN`
</ParamField>

### Option 2: API Key + Linked Account

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of records per page. Maximum 100.
</ParamField>

## Example: List All MCP Servers
```javascript
async function listMCPServers(auth, options = {}) {
  const params = new URLSearchParams();
  if (options.page) params.append('page', options.page);
  if (options.limit) params.append('limit', options.limit);

  const url = `https://api.gocobalt.io/api/v2/public/mcp-servers${params.toString() ? '?' + params : ''}`;

  const headers = auth.sessionToken
    ? { 'Authorization': `Bearer ${auth.sessionToken}` }
    : { 'x-api-key': auth.apiKey, 'linked_account_id': auth.linkedAccountId };

  const response = await fetch(url, { headers });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage with API Key
const result = await listMCPServers({
  apiKey: 'your_api_key',
  linkedAccountId: 'user_12345'
});

console.log(`Found ${result.totalDocs} MCP servers`);
result.docs.forEach(server => {
  console.log(`- ${server.name}: ${server.description}`);
});
```

## Example: Paginate Through All Servers
```javascript
async function getAllMCPServers(auth) {
  const allServers = [];
  let page = 1;
  let hasMore = true;

  while (hasMore) {
    const result = await listMCPServers(auth, { page, limit: 20 });
    allServers.push(...result.docs);
    hasMore = result.hasNextPage;
    page++;
  }

  return allServers;
}

// Usage
const servers = await getAllMCPServers({
  apiKey: 'your_api_key',
  linkedAccountId: 'user_12345'
});
console.log(`Total servers: ${servers.length}`);
```
