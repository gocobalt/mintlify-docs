---
title: Get MCP Server
api: get /api/v2/public/mcp-servers/:mcp_server_id
description: Retrieve detailed information about a specific MCP server including suggestions, chains, and associated applications.
---

Retrieve the complete configuration of an MCP server including its suggestions, chains, associated applications, workflows, and actions.

<Note>
This API supports two authentication methods: Session Token OR API Key with linked account ID.
</Note>

## Authentication

### Option 1: Session Token

<ParamField header="Authorization" type="string" required>
  Bearer token using your session token.
  
  **Format:** `Bearer YOUR_SESSION_TOKEN`
</ParamField>

### Option 2: API Key + Linked Account

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
</ParamField>

## Path Parameters

<ParamField path="mcp_server_id" type="string" required>
  Unique identifier for the MCP server.
  
  **Example:** `686e2393750d509493c6d87c`
</ParamField>

## Example: Get MCP Server Details
```javascript
async function getMCPServer(auth, mcpServerId) {
  const headers = auth.sessionToken
    ? { 'Authorization': `Bearer ${auth.sessionToken}` }
    : { 'x-api-key': auth.apiKey, 'linked_account_id': auth.linkedAccountId };

  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/mcp-servers/${mcpServerId}`,
    { headers }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage
const server = await getMCPServer(
  { apiKey: 'your_api_key', linkedAccountId: 'user_12345' },
  '686e2393750d509493c6d87c'
);

console.log(`Server: ${server.name}`);
console.log(`Suggestions: ${server.suggestions.length}`);
console.log(`Chains: ${server.chains.length}`);
console.log(`Apps: ${server.associated_apps.map(a => a.slug).join(', ')}`);
```

## Example: List Available Actions
```javascript
async function getAvailableActions(auth, mcpServerId) {
  const server = await getMCPServer(auth, mcpServerId);

  const actions = [];
  for (const app of server.associated_apps) {
    for (const action of app.actions) {
      actions.push({
        app: app.slug,
        actionId: action.id,
        name: action.name,
        description: action.description
      });
    }
  }

  return actions;
}

// Usage
const actions = await getAvailableActions(
  { apiKey: 'your_api_key', linkedAccountId: 'user_12345' },
  '686e2393750d509493c6d87c'
);

actions.forEach(action => {
  console.log(`[${action.app}] ${action.name}: ${action.description}`);
});
```
