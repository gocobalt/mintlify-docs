---
title: Update MCP Server
api: put /api/v2/public/mcp-servers/:mcp_server_id
description: Update an MCP server's configuration including suggestions, chains, and associated applications.
---

Update an existing MCP server with new configurations. You can modify the name, description, suggestions, chains, and associated applications.

<Note>
This API supports two authentication methods: Session Token OR API Key with linked account ID.
</Note>

## Authentication

### Option 1: Session Token

<ParamField header="Authorization" type="string" required>
  Bearer token using your session token.
  
  **Format:** `Bearer YOUR_SESSION_TOKEN`
</ParamField>

### Option 2: API Key + Linked Account

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
</ParamField>

## Path Parameters

<ParamField path="mcp_server_id" type="string" required>
  Unique identifier for the MCP server to update.
  
  **Example:** `686e2393750d509493c6d87c`
</ParamField>

## Example: Update MCP Server
```javascript
async function updateMCPServer(auth, mcpServerId, updates) {
  const headers = {
    'Content-Type': 'application/json',
    ...(auth.sessionToken
      ? { 'Authorization': `Bearer ${auth.sessionToken}` }
      : { 'x-api-key': auth.apiKey, 'linked_account_id': auth.linkedAccountId })
  };

  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/mcp-servers/${mcpServerId}`,
    {
      method: 'PUT',
      headers,
      body: JSON.stringify(updates)
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Add a new chain
const updated = await updateMCPServer(
  { apiKey: 'your_api_key', linkedAccountId: 'user_12345' },
  '686e2393750d509493c6d87c',
  {
    chains: [
      {
        name: 'Order Processing',
        goals: 'Process customer orders through ERP',
        steps: 'Step 1: Validate order\nStep 2: Check inventory\nStep 3: Create PO',
        precheck: 'Verify customer exists',
        successCondition: 'Order confirmed',
        responseFormat: 'JSON with order ID',
        onError: 'Escalate to support'
      }
    ]
  }
);

console.log(`Updated server with ${updated.chains.length} chains`);
```

## Example: Add Application to Server
```javascript
async function addApplicationToServer(auth, mcpServerId, appConfig) {
  // Get current configuration
  const server = await getMCPServer(auth, mcpServerId);

  // Add new application
  const associatedApps = [...server.associated_apps, appConfig];

  // Update server
  return updateMCPServer(auth, mcpServerId, { associated_apps: associatedApps });
}

// Usage
const updated = await addApplicationToServer(
  { apiKey: 'your_api_key', linkedAccountId: 'user_12345' },
  '686e2393750d509493c6d87c',
  {
    slug: 'salesforce',
    workflows: [
      { id: 'workflow_123', name: 'Lead Sync', description: 'Sync leads to Salesforce' }
    ],
    actions: [
      { id: 'create_lead', name: 'Create Lead', description: 'Create a new lead' },
      { id: 'update_contact', name: 'Update Contact', description: 'Update contact info' }
    ]
  }
);
```
