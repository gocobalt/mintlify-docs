---
title: List Webhooks
api: get /api/v2/public/webhook
description: Retrieve all registered webhooks for your organization.
---

Retrieve a list of all webhook subscriptions registered for your organization. Use this to audit webhook configurations or find webhook IDs for management operations.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Example: List and Filter Webhooks
```javascript
async function listWebhooks(apiKey) {
  const response = await fetch('https://api.gocobalt.io/api/v2/public/webhook', {
    headers: {
      'x-api-key': apiKey
    }
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Get all webhooks and filter
const webhooks = await listWebhooks('your_api_key');

// Filter active webhooks
const activeWebhooks = webhooks.filter(w => w.is_enabled);
console.log(`Active webhooks: ${activeWebhooks.length}`);

// Filter by event type
const errorWebhooks = webhooks.filter(w => 
  w.webhook_events.includes('Workflow Errored')
);
console.log(`Webhooks monitoring errors: ${errorWebhooks.length}`);
```

## Example: Webhook Status Dashboard
```javascript
async function getWebhookSummary(apiKey) {
  const webhooks = await listWebhooks(apiKey);
  
  const summary = {
    total: webhooks.length,
    active: webhooks.filter(w => w.is_enabled).length,
    disabled: webhooks.filter(w => !w.is_enabled).length,
    byType: {
      default: webhooks.filter(w => w.webhook_type === 'default').length,
      slack: webhooks.filter(w => w.webhook_type === 'slack').length
    },
    eventCoverage: {
      'Connection Created': webhooks.filter(w => w.webhook_events.includes('Connection Created')).length,
      'Connection Deleted': webhooks.filter(w => w.webhook_events.includes('Connection Deleted')).length,
      'Connection Expired': webhooks.filter(w => w.webhook_events.includes('Connection Expired')).length,
      'Workflow Completed': webhooks.filter(w => w.webhook_events.includes('Workflow Completed')).length,
      'Workflow Errored': webhooks.filter(w => w.webhook_events.includes('Workflow Errored')).length
    }
  };

  return summary;
}

// Usage
const summary = await getWebhookSummary('your_api_key');
// Output: { total: 2, active: 1, disabled: 1, byType: { default: 1, slack: 1 }, eventCoverage: {...} }
```
