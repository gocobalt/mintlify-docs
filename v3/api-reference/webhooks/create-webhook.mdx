---
title: Create Webhook
api: post /api/v2/public/webhook
description: Register a new webhook to receive real-time notifications for platform events.
---

Create a webhook subscription to receive real-time notifications when events occur in Refold. Webhooks push data to your application immediately when changes happen, eliminating the need to poll for updates.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Body Parameters

<ParamField body="webhook_url" type="string" required>
  The HTTPS URL where webhook events will be sent. Must be a valid URL that can receive POST requests.
  
  **Example:** `https://api.yourapp.com/webhooks/refold`
</ParamField>

<ParamField body="webhook_events" type="array" required>
  Array of event types to subscribe to.
  
  **Accepted values:**
  - `Connection Created` - When a linked account connects an integration
  - `Connection Deleted` - When a linked account disconnects an integration
  - `Connection Expired` - When an integration's authentication expires
  - `Workflow Completed` - When a workflow execution succeeds
  - `Workflow Errored` - When a workflow execution fails
</ParamField>

<ParamField body="is_enabled" type="boolean" default="true">
  Whether the webhook is active. Set to `false` to create a disabled webhook.
</ParamField>

<ParamField body="webhook_type" type="string" default="default">
  The webhook payload format.
  
  **Accepted values:**
  - `default` - Standard JSON payload
  - `slack` - Slack-compatible message format
</ParamField>

## Example: Register Webhook Programmatically
```javascript
async function createWebhook(webhookUrl, events, apiKey) {
  const response = await fetch('https://api.gocobalt.io/api/v2/public/webhook', {
    method: 'POST',
    headers: {
      'x-api-key': apiKey,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      webhook_url: webhookUrl,
      webhook_events: events,
      is_enabled: true,
      webhook_type: 'default'
    })
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Subscribe to all events
const webhook = await createWebhook(
  'https://api.yourapp.com/webhooks/refold',
  [
    'Connection Created',
    'Connection Deleted',
    'Workflow Completed',
    'Workflow Errored',
    'Connection Expired'
  ],
  'your_api_key'
);

console.log('Webhook created:', webhook._id);
```

<Info>
When receiving webhook events, respond with HTTP 200 OK to confirm delivery. Refold retries failed deliveries up to 3 times with exponential backoff.
</Info>
