---
title: Unsubscribe from Events (Deprecated)
api: post /api/v2/public/webhook/unsubscribe
description: Unsubscribe from webhook events. This endpoint is deprecated.
---

<Warning>
**This API is deprecated.** Use [Update Webhook](/v3/api-reference/webhooks/update-webhook) to modify subscribed events, or [Delete Webhook](/v3/api-reference/webhooks/delete-webhook) to remove the webhook entirely.
</Warning>

Unsubscribe from specific webhook events. This endpoint removes events from your current webhook subscription.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Body Parameters

<ParamField body="webhook_event" type="string" required>
  The event type to unsubscribe from.
  
  **Accepted values (legacy format):**
  - `connection-created`
  - `connection-deleted`
  - `workflow-completed`
  - `workflow-errored`
</ParamField>

## Migration Guide

Replace this deprecated endpoint with the [Update Webhook](/v3/api-reference/webhooks/update-webhook) endpoint:
```javascript
// Old way (deprecated)
await fetch('https://api.gocobalt.io/api/v2/public/webhook/unsubscribe', {
  method: 'POST',
  headers: { 'x-api-key': apiKey, 'Content-Type': 'application/json' },
  body: JSON.stringify({ webhook_event: 'connection-created' })
});

// New way (recommended)
// First, get the webhook to see current events
const webhook = await fetch(
  `https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`,
  { headers: { 'x-api-key': apiKey } }
).then(r => r.json());

// Remove the event you want to unsubscribe from
const newEvents = webhook.webhook_events.filter(e => e !== 'Connection Created');

// Update the webhook
await fetch(`https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`, {
  method: 'PUT',
  headers: { 'x-api-key': apiKey, 'Content-Type': 'application/json' },
  body: JSON.stringify({ webhook_events: newEvents })
});
```
