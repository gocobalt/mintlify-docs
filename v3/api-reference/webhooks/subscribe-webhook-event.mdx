---
title: Subscribe to Events (Deprecated)
api: post /api/v2/public/webhook/subscribe
description: Subscribe to webhook events. This endpoint is deprecated.
---

<Warning>
**This API is deprecated.** Use [Create Webhook](/v3/api-reference/webhooks/create-webhook) instead to register new webhook subscriptions.
</Warning>

Subscribe to webhook events for an existing webhook URL. This endpoint adds events to your current webhook subscription.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Body Parameters

<ParamField body="webhook_events" type="array" required>
  Array of event types to subscribe to.
  
  **Accepted values (legacy format):**
  - `connection-created`
  - `connection-deleted`
  - `workflow-completed`
  - `workflow-errored`
</ParamField>

## Migration Guide

Replace this deprecated endpoint with the [Create Webhook](/v3/api-reference/webhooks/create-webhook) endpoint:
```javascript
// Old way (deprecated)
await fetch('https://api.gocobalt.io/api/v2/public/webhook/subscribe', {
  method: 'POST',
  headers: { 'x-api-key': apiKey, 'Content-Type': 'application/json' },
  body: JSON.stringify({ webhook_events: ['workflow-errored'] })
});

// New way (recommended)
await fetch('https://api.gocobalt.io/api/v2/public/webhook', {
  method: 'POST',
  headers: { 'x-api-key': apiKey, 'Content-Type': 'application/json' },
  body: JSON.stringify({
    webhook_url: 'https://mysite.com/cobalt/webhook',
    webhook_events: ['Workflow Errored'],
    is_enabled: true,
    webhook_type: 'default'
  })
});
```
