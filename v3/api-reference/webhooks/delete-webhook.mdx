---
title: Delete Webhook
api: delete /api/v2/public/webhook/:id
description: Remove a registered webhook subscription.
---

Delete a webhook subscription. Once deleted, no further events will be sent to the webhook URL. This action is permanent.

<Warning>
Deleting a webhook is permanent. If you want to temporarily stop receiving events, consider disabling the webhook instead using the Update Webhook endpoint.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique webhook identifier to delete.
  
  **Example:** `66e5701971ff81574d55b2ba`
</ParamField>

## Example: Delete Webhook with Confirmation
```javascript
async function deleteWebhook(webhookId, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`,
    {
      method: 'DELETE',
      headers: {
        'x-api-key': apiKey
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  // Returns remaining webhooks
  return response.json();
}

// Usage: Delete and verify
async function removeWebhook(webhookId, apiKey) {
  // First, get current webhook to confirm
  const webhook = await fetch(
    `https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`,
    { headers: { 'x-api-key': apiKey } }
  ).then(r => r.json());

  console.log(`Deleting webhook: ${webhook.webhook_url}`);

  // Delete the webhook
  const remainingWebhooks = await deleteWebhook(webhookId, apiKey);
  
  console.log(`Remaining webhooks: ${remainingWebhooks.length}`);
  return remainingWebhooks;
}
```
