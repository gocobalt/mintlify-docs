---
title: Update Webhook
api: put /api/v2/public/webhook/:id
description: Update an existing webhook's URL, events, or status.
---

Update an existing webhook subscription. You can modify the webhook URL, change subscribed events, enable/disable the webhook, or change the payload format.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique webhook identifier.
  
  **Example:** `66e31564c0287b4136f77ef8`
</ParamField>

## Body Parameters

<ParamField body="webhook_url" type="string">
  The HTTPS URL where webhook events will be sent.
  
  **Example:** `https://api.yourapp.com/webhooks/refold`
</ParamField>

<ParamField body="webhook_events" type="array">
  Array of event types to subscribe to.
  
  **Accepted values:**
  - `Connection Created`
  - `Connection Deleted`
  - `Connection Expired`
  - `Workflow Completed`
  - `Workflow Errored`
</ParamField>

<ParamField body="is_enabled" type="boolean">
  Whether the webhook is active. Set to `false` to pause webhook delivery.
</ParamField>

<ParamField body="webhook_type" type="string">
  The webhook payload format: `default` or `slack`.
</ParamField>

## Example: Toggle Webhook Status
```javascript
async function toggleWebhook(webhookId, enabled, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        is_enabled: enabled
      })
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Disable webhook during maintenance
await toggleWebhook('66e31564c0287b4136f77ef8', false, 'your_api_key');

// Re-enable after maintenance
await toggleWebhook('66e31564c0287b4136f77ef8', true, 'your_api_key');
```

## Example: Update Subscribed Events
```javascript
async function updateWebhookEvents(webhookId, events, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/webhook/${webhookId}`,
    {
      method: 'PUT',
      headers: {
        'x-api-key': apiKey,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        webhook_events: events
      })
    }
  );

  return response.json();
}

// Usage: Subscribe only to error events
await updateWebhookEvents(
  '66e31564c0287b4136f77ef8',
  ['Workflow Errored', 'Connection Expired'],
  'your_api_key'
);
```
