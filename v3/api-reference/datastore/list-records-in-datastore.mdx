---
title: List Datastore Records
api: get /api/v2/public/datastore/config/:slug/:datastore/records
description: Retrieve all records from a specific config datastore.
---

Retrieve all records from a specified config datastore. Each record contains a unique `record_id` along with the columns defined when the datastore was created.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Developer → Setup**.
</ParamField>

<ParamField header="linked_account_id" type="string" required>
  The unique identifier for the linked account.
  
  **Example:** `user_12345`
</ParamField>

## Path Parameters

<ParamField path="slug" type="string" required>
  The application slug.
  
  **Examples:** `hubspot`, `salesforce`, `netsuite`
</ParamField>

<ParamField path="datastore" type="string" required>
  The name of the datastore.
  
  **Example:** `customer_sync_store`
</ParamField>

## Example: Fetch and Process Records
```javascript
async function getDatastoreRecords(linkedAccountId, slug, datastoreName, apiKey) {
  const response = await fetch(
    `https://api.gocobalt.io/api/v2/public/datastore/config/${slug}/${datastoreName}/records`,
    {
      headers: {
        'x-api-key': apiKey,
        'linked_account_id': linkedAccountId
      }
    }
  );

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  return response.json();
}

// Usage: Get all sync state records
const syncRecords = await getDatastoreRecords(
  'user_12345',
  'netsuite',
  'customer_sync_store',
  'your_api_key'
);

// Process records
const pendingSync = syncRecords.filter(r => r.sync_status === 'pending');
console.log(`Found ${pendingSync.length} records pending sync`);
```

## Example: Build Sync Status Dashboard
```javascript
async function getSyncStatusSummary(linkedAccountId, slug, datastoreName, apiKey) {
  const records = await getDatastoreRecords(linkedAccountId, slug, datastoreName, apiKey);
  
  const summary = records.reduce((acc, record) => {
    const status = record.sync_status || 'unknown';
    acc[status] = (acc[status] || 0) + 1;
    return acc;
  }, {});

  return {
    totalRecords: records.length,
    statusBreakdown: summary,
    lastUpdated: records.length > 0 
      ? records.sort((a, b) => new Date(b.last_sync) - new Date(a.last_sync))[0].last_sync
      : null
  };
}

// Usage
const summary = await getSyncStatusSummary(
  'user_12345',
  'netsuite',
  'customer_sync_store',
  'your_api_key'
);
// Output: { totalRecords: 150, statusBreakdown: { completed: 145, pending: 3, failed: 2 }, lastUpdated: "2025-01-28T10:30:00Z" }
```
