To setup your Insightly app in Refold for API Key authentication, you will need the following credentials from your Insightly account:

- API Key
- API URL (Pod)

### Pre-requisites

1. Insightly account with access to User Settings.

<Info>
  If you don't have access to the User Settings in Insightly, you'll need to contact your Insightly administrator to obtain the API key.
</Info>

### Creating an API Key in Insightly

To obtain your Insightly API key and acquire the above mentioned credentials, please follow the steps mentioned below:

1. Log in to your Insightly account at [https://app.insightly.com](https://app.insightly.com).
2. Click on the **Profile icon** in the top right corner of the navigation.

<img
  height="200"
  src="/images/insightly/insightly_navigation.png"
  alt="Navigation for API setup"
/>

3. Select `User Settings` from the dropdown menu.
4. Scroll down to the **API** section on the User Settings page.
5. Locate your **API Key** value displayed in this section.
6. Copy the **API Key** and store it securely.

<img
  height="200"
  src="/images/insightly/insightly_config.png"
  alt="Setting up Insightly API Key"
/>

7. Also note your **API URL** which indicates your pod (e.g., `https://api.na1.insightly.com/v3.1/`). The pod identifier (na1, au1, eu1, etc.) is important for API configuration.

<img
  height="200"
  src="/images/insightly/insightly_credentials.png"
  alt="Getting API Key Credentials"
/>

<Warning>
  API keys can read and modify all of your CRM data and should be handled very carefully. Store your API key securely and never share it publicly. If you believe your API key has been compromised, you can reset it from the User Settings page.
</Warning>

#### Configuring credentials in Refold

App settings page lets you configure the authentication settings for an `API Key` based application. For your customers to provide you authorization to access their data, they would first need to install your application. This page lets you set up your application credentials.

<img
  height="200"
  src="/images/insightly/insightly_refold.png"
  alt="Setting up application API Key credentials"
/>

Provide the acquired API Key and API URL (Pod) under Settings of the app and save it.
