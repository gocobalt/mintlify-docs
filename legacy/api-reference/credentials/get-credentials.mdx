---
title: Get Credentials
api: get /api/v2/public/linked-account/credentials/:linked_account_id/:slug
description: Get decrypted auth tokens for an integration using your API key.
---

Return the decrypted `authTokens` stored for an integration on a linked account. Each stored key is decrypted before being returned. Unlike [Get Auth Token](/v3/api-reference/credentials/get-token), this endpoint authenticates with your API key and does **not** refresh OAuth tokens — it returns exactly what is stored.

<Warning>
  The response contains live secrets. Use it server-side only — never log it or expose it to end users.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Path Parameters

<ParamField path="linked_account_id" type="string" required>
  Your unique identifier for the account.

  **Example:** `user_12345`
</ParamField>

<ParamField path="slug" type="string" required>
  The integration's app slug.

  **Example:** `hubspot`
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    Returns the decrypted token keys for the integration. Keys vary by app and auth type.

    <ResponseField name="access_token" type="string">Stored access token (OAuth apps).</ResponseField>
    <ResponseField name="refresh_token" type="string">Stored refresh token, when present.</ResponseField>
  </Tab>

  <Tab title="401 Unauthorized">
    <ResponseField name="status_code" type="integer">HTTP status code.</ResponseField>
    <ResponseField name="http_error_type" type="string">Error category.</ResponseField>
    <ResponseField name="error" type="string">Internal error code.</ResponseField>
    <ResponseField name="message" type="string">Human-readable error description.</ResponseField>
  </Tab>
</Tabs>

<ResponseExample>
```json 200
{
  "access_token": "CN...REDACTED_ACCESS_TOKEN...A8",
  "refresh_token": "REDACTED-REFRESH-TOKEN"
}
```
```json 401
{
  "status_code": 401,
  "http_error_type": "UNAUTHORIZED",
  "error": "SERVER_ERROR",
  "message": "Invalid Credentials"
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl -X GET "https://app.refold.ai/api/v2/public/linked-account/credentials/{linked_account_id}/{slug}" \
  -H "x-api-key: YOUR_API_KEY"
```
</RequestExample>
