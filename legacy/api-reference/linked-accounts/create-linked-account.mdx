---
title: Create Linked Account
api: post /api/v2/public/linked-account
description: Create a new linked account for your customer.
---

Create a new linked account to represent one of your customers. Use a unique identifier from your own system (such as a user or account ID) as the `account_id`. Optionally seed a custom default-app integration and user-defined fields at creation time.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your Refold API key. Find it in **Settings → Credentials**.
</ParamField>

## Body Parameters

<ParamField body="account_id" type="string" required>
  Your unique identifier for the account. Either `account_id` or `linked_account_id` (an alias) must be supplied.

  **Example:** `user_12345`
</ParamField>

<ParamField body="linked_account_id" type="string">
  Alias for `account_id`.
</ParamField>

<ParamField body="name" type="string">
  Display name for the account.
</ParamField>

<ParamField body="your_app" type="object">
  Seeds the org's default custom-app integration.
  <Expandable title="properties">
    <ResponseField name="auth_credentials" type="object" required>
      Key/value credential map stored on the default-app integration.
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField body="udf" type="object">
  User-defined fields as a key/value map. Stored internally as `{key_name, key_value}` pairs.
</ParamField>
