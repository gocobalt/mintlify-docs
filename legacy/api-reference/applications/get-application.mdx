---
title: Get Application
openapi: "GET /api/v2/public/application/{slug}"
description: Retrieve details of a specific application by its slug.
---

Retrieve detailed information about a single application by its slug. Returns the same shape as [List Applications](/legacy/api-reference/applications/list-applications), but as a single object — including connection status, authentication requirements, and connected-account details for the linked account.
