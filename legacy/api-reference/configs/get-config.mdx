---
title: Get Config by ID
openapi: "GET /api/v2/public/slug/{slug}/config/{configId}"
description: Get an installed config for an app and config ID.
---

Retrieve an installed config for an app and a specific `config_id`. Returns the fields and workflows for the installation. (To use the default config for a linked account, omit the ID — see [Get Config](/legacy/api-reference/configs/get-config-default).)

<ResponseExample>
```json 200
{
  "slug": "hubspot",
  "config_id": "user_12345",
  "field_errors": [],
  "fields": [
    { "id": "699fec7a5bd27ed2af7243ad", "name": "Select", "field_type": "select" }
  ],
  "workflows": [
    { "id": "699d8648a81ba363578cc2f4", "name": "Sync Contacts", "enabled": true }
  ]
}
```
```json 404
{
  "status_code": 404,
  "http_error_type": "NOT_FOUND",
  "error_code": "SERVER_ERROR",
  "message": "Config with config id user_12345 not found"
}
```
</ResponseExample>

<RequestExample>
```bash cURL
curl -X GET "https://app.refold.ai/api/v2/public/slug/{slug}/config/{config_id}" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID"
```
</RequestExample>
