title: File Handler Node
---
<Info>
Files greater than 5MB can be processed using this node. If your file size is less than 5MB, then it is recommended to use the [CSV/Excel by Cobalt](https://docs.gocobalt.io/build/workflow/csv-excel) node.
</Info>
## Node Functionality

File handler node gives functionality of file operation within the workflows. It can:

- Move a file to a S3 bucket.
- Perform processing of files in chunks.
