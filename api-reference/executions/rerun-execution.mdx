---
title: Rerun Execution
openapi: "POST /api/v2/public/execution/{instanceId}/rerun"
description: Retry a failed or stopped execution using its original trigger payload.
---

Retry a failed or stopped execution using its original trigger payload. This creates and starts a new execution; the original execution is left unchanged.

<ResponseExample>
```json 200
{
  "message": "Execution queued",
  "instanceId": "6a3149efd396207fd24f7aad"
}
```
```json 404
{
  "status_code": 404,
  "http_error_type": "NOT_FOUND",
  "error_code": "RESOURCE_NOT_FOUND",
  "message": "Execution not found"
}
```
</ResponseExample>

<RequestExample>
```bash cURL (API key)
curl -X POST "https://app.refold.ai/api/v2/public/execution/{instanceId}/rerun" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "linked_account_id: YOUR_LINKED_ACCOUNT_ID"
```
```bash cURL (session token)
curl -X POST "https://app.refold.ai/api/v2/public/execution/{instanceId}/rerun" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"
```
</RequestExample>
