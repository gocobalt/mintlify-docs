---
title: List Variable Definitions
openapi: "GET /api/v2/public/env/definitions"
description: List environment-variable definitions for the org.
---

List the environment-variable *definitions* for your organization, paginated. A definition describes a variable's name, scope, format, and whether it's encrypted — the actual values are managed separately (see [Set Variable Value](/api-reference/environment-variables/set-value)).
